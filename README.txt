Final model weights can be found in folder Final. All models are named according to the network.
For Two Stream Fusion, the model TwoStream.h5 is the two-stream model with the Average fuse layer.
The model TwoStreamCon.h5 is the two-stream model with the Concatenate fuse layer.
The model TwoStreamMax.h5 is the two-stream model with the Maximum fuse layer.
